# UFINDER
    - Video player build with react and consuming [Utube](https://youtube.com) public API.

## Installation
    - Clone the project
    - Create .env file
    - Add REACT_APP_API_KEY=your_api_key
    - Run
```bash
npm start
```

## Usage
   - Ufinder is listening on port 3000
